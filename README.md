## Requirements
- NodeJS v6
- 600MB RAM (1 GB Recommended)
- 800 Mhz CPU (1 Ghz Recommended)

## Getting Started
```bash
git clone https://github.com/yonle/deadsocket
cd deadsocket
./deadsocket wss://www.example.com
```
`ws` or `wss` is working too.
